# E-Wallet Use Case

This repository implements a simple e-wallet use case system where you can create users, top up balances, withdraw balances, and view transaction histories.

## Getting Started

### Prerequisites

- Node.js 

### Installation

1. **Clone the repository**:
    ```bash
    git clone https://github.com/ahmadrafidev/wallet-use-cases.git
    cd wallet-use-cases
    ```

2. **Run the code**:
    ```bash
    node wallet.js
    ```
